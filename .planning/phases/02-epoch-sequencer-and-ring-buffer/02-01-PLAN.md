<objective>
Create the lock-free ingestion pipeline with monotonic epoch assignment and pre-allocated event buffer.

Purpose: Phase 2 builds the entry point for all mutations. Events flow through the sequencer (which assigns globally unique epochs) into a pre-allocated ring buffer. The ring buffer uses power-of-2 masking for O(1) slot computation and avoids heap allocation on the hot path.

Output: `src/sequencer.rs` (AtomicU64-based epoch counter) and `src/ring_buffer.rs` (pre-allocated lock-free buffer with wrap-around) with comprehensive tests.
</objective>

<tasks>

<task id="1" name="Create epoch sequencer and ring buffer modules" commit_prefix="feat(02-01)">
Create two new modules:

**src/sequencer.rs:**
- `EpochSequencer` struct wrapping `AtomicU64`
- `new()` starting at epoch 0
- `next(&self) -> Epoch` using `fetch_add(1, SeqCst)` returning `Epoch(previous_value)`
- `current(&self) -> Epoch` returning current value without incrementing
- Implement `Send + Sync` (automatic from AtomicU64)
- Doc comments on all public items

**src/ring_buffer.rs:**
- `RingBuffer` struct with:
  - `slots: Vec<Option<(Epoch, Event)>>` — pre-allocated to capacity
  - `capacity: usize` — must be power of 2
  - `mask: usize` — capacity - 1 for bitwise slot computation
  - `sequencer: EpochSequencer` — owned sequencer for epoch assignment
  - `write_pos: usize` — next write position
- `new(capacity: usize) -> Self` — panics if capacity is not a power of 2, pre-allocates all slots as None
- `push(&mut self, event: Event) -> Epoch` — assigns epoch via sequencer, writes to slot (write_pos & mask), increments write_pos, returns assigned epoch. No heap allocation.
- `get(&self, epoch: Epoch) -> Option<&Event>` — reads slot at (epoch.0 as usize & mask), returns Some if epoch matches stored epoch, None otherwise (handles wrap-around overwrite)
- `len(&self) -> usize` — number of events written (min of write_pos and capacity)
- `capacity(&self) -> usize` — total slot count
- Implement `Send` and `Sync` manually if needed, with safety comments
- Doc comments on all public items

**src/lib.rs:**
- Add `pub mod sequencer;` and `pub mod ring_buffer;`
- Add `pub use sequencer::EpochSequencer;` and `pub use ring_buffer::RingBuffer;`

**Cargo.toml:**
- No new dependencies needed (AtomicU64 is in std)

**Verification:**
- `cargo build` — zero errors, zero warnings
- `cargo clippy` — zero warnings
</task>
