<objective>
Implement autonomous pattern detection from the mutation stream with bitvec completion tracking and auto-promotion to full frames.

Purpose: Embryonic discovery watches for emerging graph patterns without explicit frame registration. Pattern templates define what to watch for. When edges arrive, candidates track partial pattern completion using bitvec. When completion exceeds the threshold, candidates are promoted to full frames. Stale candidates are pruned, and a per-template cap prevents unbounded growth.

Output: `src/embryonic.rs` with EmbryonicDiscovery, pattern templates, candidate tracking, observe_edge, auto-promotion, pruning, and comprehensive tests.
</objective>

<task id="1" name="Create embryonic frame discovery module" commit_prefix="feat(08-01)">
Create `src/embryonic.rs`:

**PatternTemplate struct:**
- `id: u64` — unique template ID
- `pattern: Vec<HopSpec>` — multi-hop pattern to watch for
- `threshold: f64` — completion ratio required for promotion (0.0-1.0)
- `max_candidates: usize` — cap per template
- `stale_window: u64` — epochs without progress before pruning

**Candidate struct:**
- `anchor: NodeId` — candidate anchor node
- `template_id: u64` — which template this candidate is for
- `completion: BitVec` — one bit per hop, set when hop is observed (use bitvec crate)
- `last_progress_epoch: Epoch` — last epoch when a bit was set
- `created_epoch: Epoch` — when candidate was created

**PromotedFrame struct:**
- `anchor: NodeId`
- `pattern: Vec<HopSpec>`
- `template_id: u64`

**EmbryonicDiscovery struct:**
- `templates: HashMap<u64, PatternTemplate>`
- `candidates: HashMap<u64, Vec<Candidate>>` — template_id → candidates

**Operations:**
- `new() -> Self`
- `register_template(&mut self, template: PatternTemplate)` — add template
- `decompose_frame(pattern: &[HopSpec]) -> Vec<Vec<HopSpec>>` — generate all contiguous sub-patterns of length >= 2. E.g., [A,B,C] → [[A,B], [B,C], [A,B,C]]
- `observe_edge(&mut self, source: NodeId, target: NodeId, edge_type: TypeId, epoch: Epoch) -> Vec<PromotedFrame>` — for each template:
  1. Check if this edge could start a new candidate (matches first hop direction/type) → create candidate with bit 0 set
  2. Check existing candidates: if edge extends a candidate's partial path (matches the next unset hop), set that bit
  3. Check completion_ratio (set_bits / total_bits) >= threshold → promote, remove from candidates
  4. Return list of promoted frames
- `prune_stale(&mut self, current_epoch: Epoch)` — remove candidates where current_epoch - last_progress_epoch > stale_window
- `enforce_caps(&mut self)` — for each template, if candidates.len() > max_candidates, remove oldest candidates
- `candidate_count(&self) -> usize` — total candidates across all templates
- `template_count(&self) -> usize`

**src/lib.rs:** Add `pub mod embryonic;` and `pub use embryonic::EmbryonicDiscovery;`

**Verification:** `cargo build && cargo clippy` — zero errors, zero warnings
