<objective>
Create the in-memory property graph with adjacency-on-node storage, CRUD operations, directional neighbor queries, and property storage.

Purpose: The property graph is the primary storage layer that frames materialize from. It must support O(1) node lookup, efficient neighbor queries by direction and edge type, and cascading removal. Property keys are interned u32 IDs from Phase 1.

Output: `src/graph.rs` with Graph struct, node/edge CRUD, neighbor queries, property upsert, and comprehensive tests.
</objective>

<tasks>

<task id="1" name="Create property graph module with CRUD and queries" commit_prefix="feat(03-01)">
Create `src/graph.rs`:

**Internal structures:**
- `NodeData` struct: `type_id: TypeId`, `properties: HashMap<u32, PropertyValue>`, `outgoing: Vec<(EdgeId, NodeId, TypeId)>`, `incoming: Vec<(EdgeId, NodeId, TypeId)>`
- `EdgeData` struct: `edge_id: EdgeId`, `source: NodeId`, `target: NodeId`, `type_id: TypeId`

**Graph struct:**
- `nodes: HashMap<NodeId, NodeData>`
- `edges: HashMap<EdgeId, EdgeData>`
- `next_edge_id: u64` — auto-incrementing edge ID counter

**Node operations:**
- `add_node(&mut self, node_id: NodeId, type_id: TypeId)` — insert node, no-op if exists
- `remove_node(&mut self, node_id: NodeId) -> bool` — remove node + cascade remove all connected edges (both outgoing and incoming), update all neighbor adjacency lists
- `get_node_type(&self, node_id: NodeId) -> Option<TypeId>` — O(1) lookup
- `has_node(&self, node_id: NodeId) -> bool`
- `node_count(&self) -> usize`

**Edge operations:**
- `add_edge(&mut self, source: NodeId, target: NodeId, type_id: TypeId) -> Option<EdgeId>` — returns None if source or target missing, otherwise creates edge, updates both source.outgoing and target.incoming
- `remove_edge(&mut self, edge_id: EdgeId) -> bool` — removes edge from both endpoint adjacency lists
- `edge_count(&self) -> usize`

**Neighbor queries:**
- `neighbors(&self, node_id: NodeId, direction: Direction, edge_type: Option<TypeId>) -> Vec<(EdgeId, NodeId)>` — filter by direction and optional edge type

**Property operations:**
- `set_property(&mut self, node_id: NodeId, key: u32, value: PropertyValue) -> bool` — upsert, returns false if node doesn't exist
- `get_property(&self, node_id: NodeId, key: u32) -> Option<&PropertyValue>`

**src/lib.rs:** Add `pub mod graph;` and `pub use graph::Graph;`

**Verification:** `cargo build && cargo clippy` — zero errors, zero warnings
</task>
