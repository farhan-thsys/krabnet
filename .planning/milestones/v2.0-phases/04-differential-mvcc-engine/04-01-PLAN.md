<objective>
Implement the differential MVCC collection with mathematically exact +1/-1 multiset semantics, temporal snapshots, and compaction.

Purpose: The DiffCollection is the mathematical core of Krabnet. Frames store their materialized paths as differential tuples. The +1/-1 math must be provably exact — assertion + retraction = annihilation, double-assert = multiplicity 2. Temporal snapshots enable historical queries. Compaction reclaims memory without losing correctness.

Output: `src/diff.rs` with DiffCollection<T>, assert/retract operations, net delta computation, temporal snapshots, compaction, and exhaustive tests.
</objective>

<tasks>

<task id="1" name="Create differential MVCC collection" commit_prefix="feat(04-01)">
Create `src/diff.rs`:

**DiffCollection<T> struct** (where T: Clone + PartialEq + Eq + Hash):
- `tuples: Vec<DiffTuple<T>>` — all differential tuples
- `net_delta: i64` — cached aggregate net delta (sum of all delta values)

**Operations:**
- `new() -> Self` — empty collection
- `assert_tuple(&mut self, data: T, epoch: Epoch)` — push DiffTuple { data, epoch, delta: Delta(1) }, update net_delta += 1
- `retract_tuple(&mut self, data: T, epoch: Epoch)` — push DiffTuple { data, epoch, delta: Delta(-1) }, update net_delta -= 1
- `net_delta_for(&self, data: &T) -> i64` — sum of deltas for all tuples matching data (mathematically exact)
- `aggregate_net_delta(&self) -> i64` — return cached net_delta
- `snapshot(&self, epoch: Epoch) -> Vec<&T>` — return unique payloads with positive net delta from tuples at or before epoch
- `current_state(&self) -> Vec<&T>` — snapshot at Epoch(u64::MAX)
- `compact(&mut self, frontier: Epoch) -> CompactionResult` — for tuples at or before frontier: group by payload, sum deltas per payload. If sum == 0: annihilate (remove all). If sum > 0: collapse to single tuple with that sum. If sum < 0: warn, keep collapsed. Tuples after frontier unchanged. Return CompactionResult { annihilated, collapsed, warnings }.
- `tuple_count(&self) -> usize` — total tuples stored
- `is_empty(&self) -> bool`

**CompactionResult struct:**
- `annihilated: usize` — count of payloads that netted to zero
- `collapsed: usize` — count of payloads collapsed to single tuple
- `warnings: Vec<String>` — negative net delta warnings

**src/lib.rs:** Add `pub mod diff;` and `pub use diff::DiffCollection;`

**Verification:** `cargo build && cargo clippy` — zero errors, zero warnings
