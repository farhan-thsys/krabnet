<objective>
Wire all components into a single Engine struct that executes the full ingest-update-maintain-interpret pipeline end to end.

Purpose: The Engine is the top-level orchestrator. When an event arrives: (1) push to ring buffer with epoch assignment, (2) apply mutation to property graph, (3) query inverted index for affected frames, (4) for each affected frame run Tier 1 check, (5) for EdgeAdded events trigger embryonic observation, (6) auto-promote candidates to frames. Plus: frame registration, compact_all, stats reporting.

Output: `src/engine.rs` with Engine struct and comprehensive integration tests covering the full pipeline.
</objective>

<task id="1" name="Create engine module wiring all components" commit_prefix="feat(09-01)">
Create `src/engine.rs`:

**EngineStats struct:**
- `node_count: usize`
- `edge_count: usize`
- `frame_count: usize`
- `hot_frames: usize`
- `warm_frames: usize`
- `cold_frames: usize`
- `total_tuples: usize`
- `embryonic_candidates: usize`
- `embryonic_templates: usize`

**Engine struct:**
- `ring_buffer: RingBuffer`
- `graph: Graph`
- `index: InvertedIndex`
- `frames: HashMap<u64, Frame>`
- `embryonic: EmbryonicDiscovery`
- `tier_config: TierConfig`
- `previous_deltas: HashMap<u64, i64>` — previous net_delta per frame for Tier 1 checks
- `next_frame_id: u64`
- `current_epoch: Epoch`

**Operations:**
- `new(ring_buffer_capacity: usize) -> Self` — create with empty graph, index, no frames
- `ingest(&mut self, event: Event) -> Epoch` — the full pipeline:
  1. Push event to ring_buffer → get epoch
  2. Apply event to graph (match on event variant: NodeAdded → graph.add_node, NodeRemoved → graph.remove_node, EdgeAdded → graph.add_edge, EdgeRemoved → graph.remove_edge, PropertyChanged → graph.set_property)
  3. Query index.affected_frames(&event)
  4. For each affected frame: run tier1_check (compare previous_deltas entry with current frame net_delta), update previous_deltas
  5. If event is EdgeAdded: call embryonic.observe_edge → for each promoted: create Frame, materialize, register in index
  6. Update current_epoch
  7. Return epoch
- `register_frame(&mut self, anchor: NodeId, pattern: Vec<HopSpec>, epoch: Epoch) -> u64` — create Frame, materialize against graph, extract node_ids and edge_keys from materialized paths, register in inverted index, store in frames map, return frame_id
- `register_template(&mut self, template: PatternTemplate)` — delegate to embryonic
- `compact_all(&mut self, frontier: Epoch)` — compact every frame below frontier
- `query_frame(&mut self, frame_id: u64) -> Option<Vec<Vec<NodeId>>>` — query frame, return owned paths (clone from refs)
- `snapshot_frame(&self, frame_id: u64, epoch: Epoch) -> Option<Vec<Vec<NodeId>>>` — snapshot at epoch
- `stats(&self) -> EngineStats` — collect stats from all components

**Helper for extracting index keys from frame paths:**
- After materialization, collect all unique NodeIds from all paths → node_ids for index
- For consecutive node pairs in paths with known edge types → edge_keys for index
- Simpler approach: just register all NodeIds from materialized paths as node_to_frames entries

**src/lib.rs:** Add `pub mod engine;` and `pub use engine::Engine;`
